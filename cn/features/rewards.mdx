Space 奖励 [maker](/cn/concepts/makers-takers) 通过下限价单和保持市场平衡来提供流动性。市场的好坏取决于支撑其开放的资本——没有深度订单簿，价格就会变得不可靠且容易被操纵。
![Rewards Pn](/images/rewards.png)
## 运作原理
下达[限价单](/cn/guides/limit-order)会自动让您参与流动性奖励。您的奖励分配通过以下因素综合计算：
- 距离市场结算的时间（时长乘数）
- 与平衡价格的距离（质量乘数）
- 您的订单相对于市场深度的规模
没有特殊合约或隐藏机制——只需下达限价单，在您的资本提供市场深度的同时赚取奖励。您的订单帮助建立的价格信号越清晰，您赚得越多。Space 的奖励参数旨在维持所有市场的高效资本分配，确保重要但不太热门的问题仍能吸引可靠定价所需的流动性。\
\
为了保持所有市场和时间框架的价格诚实，流动性奖励根据两个关键因素分配，这两个因素决定了您对价格发现贡献的价值：
## 1. 基于时长的奖励
期限较长的市场获得更高的奖励乘数。如流动性奖励乘数曲线所示，结算时间较远的市场获得逐步更高的奖励，乘数在前 90 天显著增加，并在全年市场中继续上升。\
\
这解决了一个根本性挑战：提供者自然偏好资本不被锁定的较短时间框架。通过在期限较长的市场提供更高奖励，流动性被引导到最需要但自然最稀缺的地方。6 个月后结算的市场比下周结算的市场赚取更多——补偿您延长的资本承诺。

## 2. 基于质量的奖励
并非所有流动性都同样有价值。承诺质量曲线显示，在 \$0.50 平衡价格附近下达的订单获得最高奖励，而在极端价格（\$0.05 或 \$0.95）的订单获得最低奖励。\
\
为什么？\$0.50 附近的订单代表最大不确定性——真正的价格发现发生的地方。这些是最难维持的头寸，因为两个结果都没有明显优势，使您的资本最脆弱但对市场功能也最有价值。您正好在分歧存在和准确定价最重要的地方提供流动性。\
\
极端价格（\$0.05 或 \$0.95）的订单是更容易的承诺——一个结果似乎很明显，所以风险较小但价格发现价值也较小。这些订单获得相应较低的奖励，因为它们对找到真正的市场共识贡献较少。

Space 动态费用曲线是为 [maker 和 taker](/cn/concepts/makers-takers) 设计的激励对齐机制，通过费用不对称来实现以下目标：

- 奖励早期参与、坚定信念和冒险精神
- 维护有序的订单簿
- 奖励提供更深流动性的 maker（参见：[奖励](/cn/features/rewards)）
- 激励持续、长期的参与
- 惩罚市场操纵

<Note>
  通过限价单向订单簿添加流动性的交易者（maker）支付 0 费用。以下费用仅适用于立即成交的市价单（taker）
</Note>

1. **买入费用**：当通过市价单开仓时，费用在市场不确定阶段（您的赔率最高时）开始较高，随着市场确定性增加而逐渐降低

$$
F_{\text{buy}} (p) = F_{\text{min}} + (F_{\text{max}} - F_{\text{min}}) \times (1-(p/100)^{\alpha})
$$

其中

- $F_{\text{buy}} (p)$：概率 $p \%$ 时的买方费用
- $F_{\text{max}} = 2.00\%$：$0\%$ 概率时的最高费用（最高不确定性）
- $F_{\text{min}} = 0.02\%$：$100\%$ 概率时的最低费用（确定性）
- $\alpha = 1.3$：曲线形状系数（控制下降的平滑度）\
  \
  **示例：** \
  在 $p = 20\%$ 时，$F_{\text{buy}}(20) \approx 1.76\%$，但您购买的合约可能支付 \$1（400% ROI），所以您的赔率更好 \
  在 $p = 80\%$ 时，$F_{\text{buy}}(80) \approx 0.10\%$，您将支付 0.05% 的买入费用，但您的上涨空间较小，为 \$0.25（25% ROI），因为您的赔率较差

2. **卖出费用：** 如果您等到市场结算或下达未立即触发的限价单退出，您就是 maker，因此不支付任何费用。但是，如果您想通过市价单提前平仓，您需要了解费用在中点附近达到峰值（在 50% 概率或 \$0.50 时），此时不确定性和波动性最高；YES 或 NO 双方都没有明显的信息优势。这是价格操纵和短期套利最有可能发生的点，因此为了保护市场完整性和稳定定价，费用最高（1%），并随着结果变得更加确定而向两侧逐渐降低（0.02%）。

   $$
   F_{\text{sell}} (p) = F_{\text{min}} + (F_{\text{peak}} - F_{\text{min}}) \times e^{-0.5 (\frac{p-50}{\sigma})^2}
   $$

   其中
   - $F_{\text{sell}} (p)$：概率 $p \%$ 时的卖方费用
   - $F_{\text{peak}} = 1.00\%$：$50\%$ 概率时的最高费用（最大不确定性）
   - $F_{\text{min}} = 0.02\%$：$100\%$ 概率时的最低费用（确定性）
   - $\alpha = 1.3$：曲线形状系数（控制下降的平滑度）

   在 \$0.50（50% 概率）时，您的卖出费用是 1%

   在 \$0.20（20% 概率）时，您的卖出费用约为 0.12%\
   \
   **示例：**\
   您希望通过市价单进行以下交易（如果您通过限价单进入，则不需要支付任何费用）。

|                                | **数值**                                       |
| -----------------------------: | ---------------------------------------------- |
| **订单类型**                   | 市价单（收取费用；限价单无费用）               |
| **市场**                       | "比特币今年会收于 150,000 美元以上吗？"        |
| **当前比特币价格**             | \$125,000                                      |
| **当前市场概率**               | 30%                                            |
| **购买份额**                   | 1,000 份 YES，每份 \$0.30                      |

<Steps>
  <Step title="开仓" titleSize="h3">
    - 头寸规模：1,000 x \$0.30 = \$300
    - 买入费用 ≈ 1.6%（\$4.80）
    - **总成本：** \$304.80
  </Step>
  <Step title="如果您希望在市场结算前平仓" titleSize="h3">
    - 比特币上涨至 \$145，市场概率升至 80%
    - 卖出价格：\$0.80 x 1,000 = \$800
    - 卖出费用 ≈ 0.04%（\$0.32）
    - 净收益：\$799.68
    - **净利润：** \$800 - \$0.32 - \$304.80 = \$494.88（162% ROI）
  </Step>
</Steps>

杠杆通过 [CLOB](/cn/architecture/clob) 上的保证金交易放大敞口。您存入抵押品来控制比保证金正常允许更大的头寸。
## 运作原理
|                  | **数值**     |
| ---------------: | ------------ |
| **保证金**       | \$100        |
| **杠杆**         | 5×           |
| **头寸规模**     | \$500        |
| **\$0.15 的份额** | 3,333 份     |
如果价格 = \$0.30：+\$500 利润（500% ROE）
如果价格 ≈ \$0.13：触发清算
## 关键参数
- **初始保证金（IM）：** 20%（开仓所需）
- **维持保证金（MM）：** 10%（保持头寸所需）
- **最大杠杆：** 10 倍
## 头寸规模计算
$ \text{头寸规模} = \text{保证金} \times \text{杠杆}$
$\text{份额} = \text{头寸规模} / \text{入场价格}$
**示例：**
$$
\$100 \ \text{保证金} \times 5 = \$500 \ \text{头寸}
$$
$$
\$500 / \$0.15 = 3,333 \ \text{份额}
$$
## 清算价格
$$
\frac{(\text{入场价格} \times \text{份额}) - \text{保证金}}{\text{份额} \times (1 - MM)}
$$
**示例：**
|                | **数值**                                                                           |
| -------------: | ---------------------------------------------------------------------------------- |
| **入场价格**   | \$0.15                                                                             |
| **保证金**     | \$100                                                                              |
| **份额**       | 3,333                                                                              |
| **清算价格**   | ≈\$0.13                                                                            |
| **说明**       | 如果市场价格 ≈\$0.13，您的权益等于头寸价值的 10%（\$44.39），清算开始。             |

## 盈亏计算
$$
\text{盈亏} = (\text{退出价格} - \text{入场价格}) \times \text{份额}
$$
**示例：**
|            | 数值                                    |
| ---------- | --------------------------------------- |
| **退出**   | \$0.30                                  |
| **入场**   | \$0.15                                  |
| **份额**   | 3,333                                   |
| **盈亏**   | (\$0.30 - \$0.15) × 3,333 = **\+\$500** |
| **ROE**    | \$500 / \$100 = **500%**                |
杠杆按比例放大收益和损失。
**示例：**
|            | 数值                                  |
| ---------- | ------------------------------------- |
| **退出**   | \$0.13                                |
| **入场**   | \$0.15                                |
| **份额**   | 3,333                                 |
| **盈亏**   | (\$0.13 - \$0.15) × 3,333 = **-\$67** |
| **ROE**    | -\$67 / \$100 = **-67%**              |

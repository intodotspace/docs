\$SPACE는 시즌별 에어드롭을 통해 사용자에게 배포됩니다. 귀하의 몫은 획득한 포인트에 따라 결정됩니다 - 포인트가 많을수록 에어드롭이 커집니다. 최고 성과자는 독점 이벤트 초대도 받습니다.
## 포인트란 무엇인가요?
포인트는 Space에서의 의미 있는 참여를 측정합니다: 거래, [지정가 주문 유동성](/ko/features/rewards) 제공, 추천, 지속적인 참여. 이는 Space가 에어드롭을 배포할 때 사용하는 감사 가능한 기록입니다. 포인트는 참여 지표이며, 토큰이나 미래 보상의 보장이 아닙니다.
포인트 계산 세부사항과 자격 요건은 각 시즌이 시작되기 전에 발표되며, 포인트는 각 새 시즌이 시작될 때 리셋됩니다.

## 포인트 획득 방법
- 거래 활동
- 추천
- 대회 참여
- 태스크 완료
- 일일 로그인 및 연속 보너스
- 업적 달성
- 커뮤니티 참여
- 소셜 플랫폼 참여
## 에어드롭
Space의 에어드롭은 배포 도구이지 경품이 아닙니다. 실제로 제품을 사용하는 사람들과 소유권을 일치시킵니다: 트레이더, 유동성 제공자, 제휴자 등이 그 예입니다. 실제 참여자에게 토큰을 배포하면 탈중앙화가 강화되고 마켓을 유동적이고 신뢰할 수 있게 만드는 행동에 보상이 주어집니다.
## 시즌이란 무엇인가요?
Space는 시즌을 통해 소유권을 배포합니다: 토큰 배포를 실제 플랫폼 사용과 일치시키는 시간 제한 보상 기간입니다. 각 시즌은 기여를 측정하고 커뮤니티 소유권을 확대합니다.
**작동 방식:**
1. **발표:** Space는 각 시즌 전에 기간, 보상 가중치, 자격 기준을 발표합니다.
2. **포인트 획득:** 거래, 참여, 추천으로 사용자 유치, 연속 유지, 또는 유동성 제공.
3. **스냅샷:** Space는 시즌 종료 시 참여 데이터를 기록합니다.
4. **배포:** 생태계 보상은 검증된 포인트에 비례하여 적격 지갑에 배포됩니다.
5. **리셋:** 포인트가 리셋되고 업데이트된 가중치와 기능으로 새 시즌이 시작됩니다.
## 시즌이 중요한 이유
소유권이 생태계 참여와 일치할 때 장기적 가치가 창출됩니다. 실제 사용자에게 광범위한 소유권을 부여하면 더 강한 네트워크 효과, 더 깊은 유동성, 더 높은 플랫폼 신뢰가 형성됩니다 - 그리고 결국 덤핑이 아닌 토큰 상승으로 이어집니다 ([Hyperliquid, 2025](https://nodescience.medium.com/ultimate-guide-on-hyperliquid-season-2-07-19-updated-bb16870f4d98)). Space는 사용자를 이해관계자로, 참여를 지분으로 전환합니다.

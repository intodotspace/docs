레버리지는 [CLOB](/ko/architecture/clob)에서 마진 거래를 통해 노출을 확대합니다. 담보를 예치하여 마진이 일반적으로 허용하는 것보다 큰 포지션을 제어합니다.
## 작동 방식
|                      | **값**       |
| -------------------: | ------------ |
| **마진**             | \$100        |
| **레버리지**         | 5×           |
| **포지션 크기**      | \$500        |
| **\$0.15 기준 주식** | 3,333 주     |
가격 = \$0.30인 경우: +\$500 수익 (500% ROE)
가격 ≈ \$0.13인 경우: 청산 발동
## 주요 파라미터
- **초기 마진 (IM):** 20% (포지션 오픈에 필요)
- **유지 마진 (MM):** 10% (포지션 유지에 필요)
- **최대 레버리지:** 10배
## 포지션 크기 계산
$ \text{포지션 크기} = \text{마진} \times \text{레버리지}$
$\text{주식 수} = \text{포지션 크기} / \text{진입 가격}$
**예시:**
$$
\$100 \ \text{마진} \times 5 = \$500 \ \text{포지션}
$$
$$
\$500 / \$0.15 = 3,333 \ \text{주}
$$
## 청산 가격
$$
\frac{(\text{진입 가격} \times \text{주식 수}) - \text{마진}}{\text{주식 수} \times (1 - MM)}
$$
**예시:**
|                | **값**                                                                              |
| -------------: | ----------------------------------------------------------------------------------- |
| **진입 가격**  | \$0.15                                                                              |
| **마진**       | \$100                                                                               |
| **주식 수**    | 3,333                                                                               |
| **청산 가격**  | ≈\$0.13                                                                             |
| **설명**       | 시장 가격이 ≈\$0.13이면, 귀하의 자기자본이 포지션 가치의 10%(\$44.39)가 되어 청산이 시작됩니다. |

## 손익 계산
$$
\text{손익} = (\text{청산 가격} - \text{진입 가격}) \times \text{주식 수}
$$
**예시:**
|            | 값                                      |
| ---------- | --------------------------------------- |
| **청산**   | \$0.30                                  |
| **진입**   | \$0.15                                  |
| **주식 수**| 3,333                                   |
| **손익**   | (\$0.30 - \$0.15) × 3,333 = **\+\$500** |
| **ROE**    | \$500 / \$100 = **500%**                |
레버리지는 이익과 손실을 비례적으로 확대합니다.
**예시:**
|            | 값                                    |
| ---------- | ------------------------------------- |
| **청산**   | \$0.13                                |
| **진입**   | \$0.15                                |
| **주식 수**| 3,333                                 |
| **손익**   | (\$0.13 - \$0.15) × 3,333 = **-\$67** |
| **ROE**    | -\$67 / \$100 = **-67%**              |
